# AI + 跨境电商人设打造规划

## 🎯 人设定位

**最终人设**：用AI做跨境电商的老秦

**人设公式**：
```
AI探索者（技术感） + 跨境实战派（接地气） = AI跨境老秦
```

**人设标签**：
- 🔧 技术流玩家（会折腾AI工具）
- 💼 跨境实战派（10年义乌经验）
- ⚡ 效率狂人（用AI10倍提效）
- 🎯 务实主义者（只分享能落地的东西）

## 📅 四阶段内容策略

### 阶段1：AI工具探索者（现在 - 第2周）

**内容方向**：
| 类型 | 占比 | 内容 |

## 📝 内容矩阵规划

### 抖音（流量入口）
**定位**：短视频钩子，引发好奇
- 时长：15-60秒
- 风格：快节奏、强冲突、有料
- 选题：踩坑、效果对比、反常识观点

### 视频号（信任建立）
**定位**：中视频讲解，建立专业度
- 时长：1-5分钟
- 风格：娓娓道来，干货密度高
- 选题：完整教程、案例拆解、深度观点

### 知乎（搜索流量）
**定位**：长文沉淀，SEO获客
- 字数：1500-3000字
- 风格：专业、结构化、有数据
- 选题：教程、方法论、行业分析

### 小红书（精准流量）
**定位**：图文笔记，工具分享
- 形式：图文 + 简短视频
- 风格：实用、美观、易收藏
- 选题：工具清单、步骤教程、避坑指南

### 朋友圈（私域运营）
**定位**：日常记录，人设维护
- 频率：每天2-3条
- 风格：真实、有温度、有互动
- 内容：日常思考、成果展示、求助互动

## 🔗 人设过渡技巧

### 阶段1→阶段2（如何自然过渡）
**方法**：在展示AI能力时，开始提"业务场景"

**示例**：
- "用AI帮我写文案" → "用AI帮我写跨境文案"
- "AI自动发消息" → "AI自动回复海外客户"
- "AI分析数据" → "AI分析竞品店铺数据"

### 阶段2→阶段3（如何叠加跨境标签）
**方法**：用AI解决的第一个跨境问题

**示例选题**：
- "跨境电商第一步：用AI分析选品数据"
- "不会写英文文案？AI帮我搞定亚马逊listing"
- "海外客户咨询，AI自动翻译+回复"

### 阶段3→阶段4（如何确立专家地位）
**方法**：输出方法论，带人出结果

**示例选题**：
- "AI跨境三步法：选品-内容-投放"
- "我带10个学员用AI做跨境，7个出单了"
- "AI跨境时代，个人卖家如何弯道超车"
