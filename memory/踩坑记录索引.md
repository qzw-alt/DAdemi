## 如何添加新的踩坑记录

1. 创建新文件：`memory/踩坑_分类_具体事项.md`
2. 遵循统一格式：
   ```markdown
   ## 坑 X：标题
   
   ### 现象
   - 发生了什么
   
   ### 原因
   - 为什么会这样
   
   ### 解决方案
   - 怎么解决
   
   ### 预防措施
   - 下次怎么避免
   ```
3. 更新本索引
4. 同步更新 MEMORY.md
