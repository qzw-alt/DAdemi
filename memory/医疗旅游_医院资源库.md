# 🏥 国际部标准就诊流程

## 1. 到院前
- 带好：证件 + 保险卡 + 预约截图 + 病历
- 商保用户：提前让保险公司发预约确认/授权号
- 记好：医院地址、国际部入口、科室

## 2. 到院后第一步：找国际部
- 不要去普通门诊大厅
- 直接去：国际医疗部 / VIP门诊 / 特需楼
- 进门先到：前台/服务台

## 3. 登记报到
- 出示：身份证/护照 + 预约信息
- 商保：出示保险卡 + 授权号
- 前台登记、核对信息、给你就诊号/候诊区

## 4. 商保直付办理（重点）
- 填写：医院直付申请表
- 前台/商保专员：拍照保险卡、授权号、证件
- 确认：本次费用直付/自付比例
- 完成后：不用现场缴费，直接去看病

## 5. 候诊 & 看诊
- 按指引到对应科室候诊
- 叫号后进入诊室
- 医生问诊、查体、开检查/处方

## 6. 检查/化验/取药
- 国际部一般有专属检查室，不用排长队
- 检查/取药：多数可直付记账，不用先交钱
- 做完检查回诊室复诊

## 7. 看完病：结算 & 离院
- 非商保：前台缴费
- 商保直付：
  - 前台打印费用清单
  - 签字确认
  - 直接离院，医院和保险公司结算
  - 需要：可开英文病历、诊断证明、发票

## ✅ 超简版流程
到国际部前台 → 登记证件+预约 → 商保办直付 → 候诊看诊 → 检查取药 → 签字离院
