## 一、什么是TK CPA？

CPA = Cost Per Action，按成交付费
- 你帮商家推广产品链接
- 客户通过你的链接完成购买/注册
- 你赚佣金（通常$5-$50/单）

**优势**：不用囤货、不用发货、不用客服
**劣势**：佣金比带货低，需要选品眼光

## 三、实操步骤

### 第1步：确定赛道

**推荐赛道（2026年）**：

| 赛道 | 佣金率 | 竞争度 | 推荐指数 |
|-----|--------|--------|---------|
| 家居收纳 | 15-25% | 中 | ⭐⭐⭐⭐⭐ |
| 3C配件 | 10-20% | 高 | ⭐⭐⭐⭐ |
| 美妆工具 | 20-30% | 中 | ⭐⭐⭐⭐ |
| 宠物用品 | 15-25% | 低 | ⭐⭐⭐⭐⭐ |
| 健身器材 | 15-25% | 低 | ⭐⭐⭐⭐ |

**选品原则**：
1. 佣金≥15%
2. 视频好做（展示型产品）
3. 解决痛点/痒点
4. 客单价$20-80

### 第2步：找产品

**渠道一：TikTok Creator Marketplace**
- 网址：creator-marketplace.tiktok.com
- 申请加入联盟
- 搜索品类，找高佣金产品

**渠道二：电商平台选品**
- Amazon（美国站）
- eBay
- AliExpress（速卖通）
- 找有特色、视觉冲击力强的产品

**渠道三：看爆款**
- 搜索：[产品品类] + "TikTok"
- 看哪些视频点赞过万
- 跟进同款或类似品

### 第3步：找素材

**来源**：
1. 国内抖音/快手热门（去重剪辑）
2. 小红书种草笔记
3. 亚马逊产品视频
4. YouTube开箱视频

**去重技巧**：
- 镜像翻转
- 调整速度（0.9x/1.1x）
- 添加滤镜/调色
- 改变背景音乐
- 删减/重新排序片段
- 添加自己的配音/文字

### 第4步：制作视频

**黄金3秒原则**：
- 开头必须抓眼球（痛点/反差/疑问）
- 不要铺垫，直接上干货

**万能模板**：

```
0-3秒：痛点切入
"你是不是也有XXX的烦恼？"

3-10秒：产品展示
"今天给大家介绍这个神器..."

10-15秒：效果演示/卖点
"用它XXX，只需要几秒钟..."

15秒：引导下单
"链接在评论区，自取"
```

**发布设置**：
- 标题：#TikTokShop #好物推荐 #产品品类
- 评论区：置顶购买链接
- 定位：可选

### 第5步：挂链接

**方式一：个人主页Bio链接**
- 用bitly或short.io缩短链接
- 追踪点击数据

**方式二：TikTok Shop联盟链接**
- 进入Creator Marketplace
- 找到产品 → Generate Link
- 挂到视频下方

**方式三：私域引导**
- 评论区："更多信息请私信"
- 引导到WhatsApp/Line成交

## 四、进阶玩法

### 1. 矩阵玩法
- 同时做3-5个账号
- 不同品类/不同市场
- 降低风险，提高爆率

### 2. 付费投放
- 有预算可以投TikTok Ads
- 测品：$50/天，跑3天
- 好的品加大投放

### 3. 私域沉淀
- 引流到WhatsApp/邮箱
- 二次转化，推荐更多产品
- 建立自己的用户池

### 4. 爆款复制
- 一个品爆了 → 批量做同类视频
- 一个品跑通 → 复制到其他账号

## 七、常见问题

**Q：账号0播放怎么办？**
A：检查IP是否稳定、作品是否有违禁词、是否被判定搬运

**Q：视频播放高但没转化？**
A：检查链接是否生效、佣金是否设置正确、引导是否到位

**Q：被封号怎么办？**
A：申诉/换IP/换设备，重新开始

**Q：需要多少粉丝才能挂链接？**
A：个人号1000粉可挂外链，TikTok Shop无粉丝要求
