## 三、加入联盟计划

### 3.1 TikTok Creator Marketplace（官方联盟）

**入口**：creator-marketplace.tiktok.com

**申请条件**：
- 账号粉丝≥1000（部分地区无要求）
- 年满18岁
- 账号内容符合TK社区规范

**申请流程**：

```
1. 访问Creator 点击"Join as a Creator"
3 Marketplace
2.. 绑定TikTok账号
4. 填写个人资料（姓名、邮箱、国家）
5. 提交申请
6. 等待审核（1-7天）

【技巧】
- 申请时附上其他社媒账号（Instagram/YouTube）
- 提高通过率
```

### 3.2 第三方联盟平台

| 平台 | 网址 | 特点 |
|-----|------|------|
| Amazon Associates | affiliates.amazon.com | 佣金高、审核严 |
| eBay Partner Network | partnernetwork.ebay.com | 品类全 |

| ShareASale | shareasale.com | 大牌商家多 |
| CJ Affiliate | cj.com | 适合美国市场 |

### 3.3 选品技巧

**高佣金品类推荐**：

| 品类 | 佣金比例 | 客单价 | 推荐理由 |
|-----|---------|-------|---------|
| 手机配件 | 15-25% | $20-50 | 刚需、易展示 |

| 美妆工具 | 25-35% | $15-40 | 复购率高 |

**选品公式**：
```
好产品 = 佣金≥15% × 视频好做 × 解决痛点 × 20<$<80
```

## 五、视频制作（爆款模板）

### 5.1 万能开头公式

**痛点开头**：
```
"你是不是也有XXX的烦恼？"
"90%的人都不知道XXX！"
"别再XXX了！"
```

**反常识开头**：
```
"XXX其实是个骗局！"
"XXX千万不要买！"
"我花了XXX刀发现的真相！"
```

**结果先行**：
```
"用这个方法，我3天赚了XXX刀"
"这个产品让我月入XXX刀"
"XXX竟然只要XXX刀！"
```

### 5.2 爆款结构（15-60秒）

```
0-3秒：Hook（抓眼球）
├─ 痛点/反常识/结果先行
└─ 配合夸张表情/音效

3-10秒：Problem（提出问题）
├─ 描述痛点
└─ 引发共鸣

10-25秒：Solution（解决方案）
├─ 展示产品
├─ 演示效果
└─ 强调卖点

25-30秒：CTA（行动号召）
├─ "点击链接查看"
└─ "评论区自取"
```

### 5.3 热门文案模板

**家居类**：
- "这个XXX让我家客厅瞬间整洁！"
- "后悔没早买！XXX解决了我XXX年的烦恼"
- "XXX块钱的XXX，竟然比XXX还好用"

**3C类**：
- "这个手机配件让我XXX效率翻倍"
- "iPhone用户必备！XXX"
- "XXX充电速度也太快了"

**美妆类**：
- "这个技巧让妆容精致10倍"
- "XXX网红同款！用了XXX天后的变化"
- "别再花冤枉钱买XXX了"

## 六、挂链接与追踪

### 6.1 链接挂载方式

**方式1：个人主页Bio（推荐新手）**

```
1. 制作短链：bitly.com 或 short.io
2. 粘贴联盟链接 → 生成短链
3. 打开TikTok → 个人主页 → 编辑资料
4. 在网站栏粘贴短链
5. 保存
```

**方式2：视频描述区**

```
1. 在视频描述中 @产品
2. 或在评论区置顶购买链接
3. 引导："Link in bio" 或 "点击下方链接"
```

**方式3：TikTok Shop联盟**

```
1. 进入Creator Marketplace
2. 搜索目标产品
3. 点击"Get Link"或"Generate Link"
4. 选择视频挂载
5. 提交
```

### 6.2 数据追踪

**必须追踪的数据**：
- 视频播放量
- 链接点击次数
- 转化数（成交单数）
- 佣金收入

**追踪工具**：
- Bitly：免费链接统计
- Google Analytics：网站流量
- TikTok Creator：后台数据

## 八、问题排查

### 8.1 常见问题

| 问题 | 原因 | 解决方案 |
|-----|------|---------|
| 0播放 | IP不稳定/被检测/内容违规 | 检查IP、换视频内容 |
| 有播放无转化 | 链接失效/佣金未生效/价格高 | 检查链接、换产品 |
| 账号被封 | 违规操作/IP问题 | 申诉或换号重来 |
| 链接点不了 | 未挂车/产品下架 | 重新生成链接 |

### 8.2 避坑清单

**❌ 绝对不要**：
1. 买粉买赞（必封）
2. 刷点击量（必封）
3. 卖违禁品（口罩、电子烟等）
4. 搬运原视频（必须去重）
5. 频繁换IP/设备

**✅ 必须做到**：
1. 一机一号一号
2. 每天发3-5条
3. 视频必须去重处理
4. 保持IP稳定
5. 遵守TK社区规则
