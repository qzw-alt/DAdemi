# 资讯库 - 索引

## 目录结构

```
资讯库/
├── AI资讯/
│   └── YYYY-MM.md    # 每月一个文件
└── 跨境电商资讯/
    └── YYYY-MM.md    # 每月一个文件
```

## 保存规则

- **只保存最近1个月的内容**
- 每月一个文件，格式：2026-02.md
- 每月1号自动清理上上月的旧内容

## 调用场景

- 09:00 AI资讯报告 → 从AI资讯库读取
- 写内容需要行业信息 → 从对应资讯库调用

## 文件格式

```markdown
# 2026-02 AI资讯

## 02-23
- 资讯标题
- 来源/日期
- 摘要要点

## 02-22
- ...
```

## 自动清理

- 每月1号执行：删除上上月文件夹
- 例：3月1日删除1月的全部资讯
