# 人设_老板学员案例

## 学员案例库（持续更新）

### 案例1：深圳电子厂张总
- 背景：内贸做了10年，2025下半年转型跨境
- 痛点：不懂英语，工厂有产品优势
- 结果：3个月出第一单，通过TikTok获取询盘
- 金句："英语不是问题，问题是敢不敢开始"

### 案例2：义乌饰品李姐
- 背景：夫妻店，线下展会受阻
- 痛点：产品利润薄，渠道单一
- 结果：Facebook+独立站双通道，月询盘30+
- 金句："小产品也能做成大生意"

### 案例3：广州服装王老板
- 背景：传统B2B，疫情后订单下滑
- 痛点：不懂社媒，获客成本高
- 结果：LinkedIn开发出首个美国大客户
- 金句："不是询盘难找，是你的内容没人看"

### 案例4：宁波五金刘总
- 背景：工厂型，之前只做代工
- 痛点：没有品牌，定价权低
- 结果：亚马逊+独立站，品牌溢价30%
- 金句："代工是帮别人赚钱，品牌才是自己的"

---

**人设口播素材方向：**
- 用真实学员故事做背书
- 强调"普通人也能学会"
- 突出"改变结果"而非"学习方法"
