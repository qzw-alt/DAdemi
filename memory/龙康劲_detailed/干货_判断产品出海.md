# 如何判断你的产品是否适合出海

## 主题
中国下一代有钱人一定是做外贸的

## 第一问：你的产品能否出海？

### 判断方法：看数据
去亚马逊、虾皮等跨境平台搜你的产品类目，重点看：

1. **销量**: 找评论超过500-1000的链接 → 说明国外真有市场
2. **利润**: 对比国外售价和出厂价
   - 中东：出厂价3倍以上
   - 北美：出厂价5倍以上
   - 这样生意才值得做

## 第二问：产品该销往哪个国家？

### 方法：投流测试
1. 在Facebook上针对一个国家建2-3条广告计划
2. 每条每天预算几十块人民币（不超过100块）
3. 跑一周，看三个数据：

**询盘量**: 当地采购商意愿强不强
**汇总问题**: 客户关心价格？样品？认证？
**转化率**: 询盘能不能成交

### 结论
- 询盘多、成本低、成交准 → 这个市场值得扎进去
- 谁说了都不算，自己去测
- 数据不会骗人
