# 外贸新手最容易踩的四个大坑

## 主题
做外贸别光想着赚钱，你得先想清楚万一你赔钱，这钱会赔在哪儿

## 四大坑

### 第一坑：网络
- **问题**: 找便宜梯子蹭共享IP，IP乱跳
- **后果**: 平台判定账号异常，辛苦养的号说没就没
- **建议**: 网络是吃饭家伙，必须稳定独享，一个月几十块钱就能搞定

### 第二坑：等流量
- **问题**: 天天光发视频等自然流
- **后果**: 流量不稳定不精准，耗几个月没结果自己就崩溃了
- **建议**: 学会精准投流，用最小成本快速试错打到询盘

### 第三坑：想走捷径
- **问题**: 相信什么包客户、黑科技软件
- **后果**: 被割韭菜，钱财两空
- **建议**: 做外贸没有捷径，核心就是产品+流量+沟通

### 第四坑：收款问题（最要命）

**假付款骗货**
- 客户发来水单，催你发货
- 不见兔子不撒鹰：银行账户查清楚，真金白银到账再发货

**乱报价**
- 客户问EXW价，你把运费也算进去 → 贵跑人
- 客户问CIF价，你只报产品成本 → 自己亏死
- **建议**: FOB、CIF、EXW这些术语必须门儿清

**信用证挖坑**
- 公司名称地址错个字母 → 几十万美金卡半年
- **建议**: 所有信息必须百分百核对，一字不错

**赊账OA**
- 给新客户账期 → 钱在哪儿都不知道
- **案例**: 义乌约旦骗子骗走16个柜的货，1000多万
- **建议**: 合作多年、知根知底的老客户才能考虑

## 总结
- 网络是腿
- 流量是路
- 防骗是脑
- 收款是命
