# 人设_常见问题FAQ

## 学员高频问题库

### Q1：英语不好能做跨境吗？
**A：** 当然能！现在翻译软件、AI工具非常强大。关键是你的产品专业度，采购商更看重的是产品质量和供应链能力。英语只是工具，又不是门槛。

### Q2：没经验从哪个平台开始？
**A：** 
- 有工厂资源 → 阿里国际站/独立站
- 懂年轻人消费品 → TikTok/亚马逊
- 工业品/定制 → LinkedIn/独立站
- **记住：平台不重要，流量在哪里就去哪里**

### Q3：需要多少资金启动？
**A：** 
- 轻资产（社媒获客）：5000-2万/年
- 平台开店（阿里/亚马逊）：3-10万
- 独立站：1-5万
- **建议从小做起，赚了再投**

### Q4：自己摸索还是找培训机构？
**A：** 
- 免费资源：YouTube、B站、知乎
- 付费学习：系统化，省时间
- **关键：学完必须执行，别做"学习控"**

### Q5：怕被骗怎么办？
**A：** 
- 任何承诺"保证出单"的都是骗子
- 正规培训机构不会保证结果
- 签合同前仔细阅读条款
- **先小金额测试，再决定是否深入**

### Q6：产品怎么选？
**A：** 
- 优先自己熟悉的产品
- 看同行在卖什么（亚马逊搜、热销榜）
- 避开巨头垄断类目
- **小品类有机会，大类目要差异化**

### Q7：做阿里还是独立站？
**A：** 
- 阿里：有自然流量，但竞争激烈，价格透明
- 独立站：需要自己引流，但利润高，品牌自主
- **建议：两者都做，阿里获客，独立站沉淀**

### Q8：多久能出第一单？
**A：** 
- 快的1-2周（平台自然流量）
- 慢的3-6个月（需要积累）
- **平均3个月是正常周期，心态要稳**
