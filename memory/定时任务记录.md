## 9个定时任务

### 1. 社区学习
- **时间**: 每天 00:00（北京时间）
- **任务**: 在Moltbook社区学习新技术
- **目标**: 保持技能更新，了解AI最新动态

### 2. 清理过期参考文案
- **时间**: 每天 03:00（北京时间）
- **任务**: 清理过期的参考文案和资料
- **目标**: 保持资料新鲜，避免信息过时

### 3. 填充技能库和文案库
- **时间**: 每天 06:00（北京时间）
- **任务**: 学习新技能，补充文案素材
- **目标**: 持续积累，为内容创作做准备

### 4. 晨间记忆读取
- **时间**: 每天 06:30（北京时间）
- **任务**: 读取以下文件，建立当天上下文
  - MEMORY.md（长期记忆要点）
  - memory/龙康劲_index.md（项目速查）
  - memory/skills记录.md（技能安装状态）
  - 前一天 memory/YYYY-MM-DD.md（工作日志）
- **目标**: 快速恢复上下文，了解昨天发生了什么
- **重要**: 每次读取后检查API配置状态

### 5. AI资讯报告
- **时间**: 每天 09:00（北京时间）
- **任务**: 
  - 搜索当天AI/跨境电商最新资讯
  - 保存到 memory/资讯库/ 对应目录
  - 自动删除超过1个月的旧资讯
- **目标**: 为伟烨提供有价值的行业信息，共同学习
- **输出**: 把收集到的内容发给伟烨看
- **格式**: memory/资讯库/AI资讯/2026-02.md

### 6. 朋友圈图文
- **时间**: 每天 10:00（北京时间）
- **任务**: 准备并发布朋友圈内容
- **目标**: 维持社媒活跃度

### 7. 知乎+小红书
- **时间**: 每天 11:00（北京时间）
- **任务**: 发布知乎/小红书内容
- **目标**: 多平台分发，扩大影响力

### 8. 总结+项目资料保存
- **时间**: 每天 22:00（北京时间）
- **任务**: 
  - 总结当天工作
  - 更新项目资料（龙康劲_detailed/、龙康劲_operations/）
  - 保存到 memory/YYYY-MM-DD.md
- **目标**: 记录当天进展，积累项目资产

### 9. 记忆备份
- **时间**: 每天 23:30（北京时间）
- **任务**: 
  - 将当天重要记忆同步到飞书云文档
  - 备份 MEMORY.md
  - 备份 skills记录.md
  - **直接执行，无需伟烨确认**
- **目标**: 防止断联丢失记忆，实现跨设备恢复

---

## 任务执行原则

1. **每日必执行**: 9个任务按时间顺序执行
2. **重要优先**: 记忆备份(23:30)是最重要的，防止丢失
3. **遇错记录**: 如果任务失败，记录错误信息到当日日记
4. **变更同步**: 任务调整后立即更新本文件
